# Stock_news_pusher
I'm a Telegram chatbot and I can get stock infomation online and push them to you via Telegram.

## Usage
#### Download and unzip
`wget -O "Stock_news.zip" https://github.com/SpereShelde/Stock_news/archive/master.zip`

`unzip Stock_news.zip`

#### Edit API_key

`nano tg_bot.py`

#### Deploy

`python3 tg_bot.py`
